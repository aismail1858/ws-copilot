# ui-research

UI/UX-Vergleiche, Design-Patterns, Usability-Studien, Interface-Analysen.

## Inhalt

- Vergleichende UI-Analysen (Screenshots, Flows, Muster)
- Design-System-Evaluation
- Usability-Test-Ergebnisse
- Zugänglichkeit (a11y) Recherche
- Prototypen-Referenzen und Inspirationen

## Hinweise

- Visuelle Ergebnisse als `.mmd`-Diagramme, wenn strukturiert darstellbar.
- Bei jeder UI-Entscheidung: Task-MD-Referenz und ADR-Verknüpfung pflegen.
