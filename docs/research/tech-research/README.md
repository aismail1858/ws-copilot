# tech-research

Technologie-Evaluierung, Framework-/Lib-Vergleiche, Prototyping-Ergebnisse, Machbarkeitsanalysen.

## Inhalt

- Vergleiche von Frameworks, Libraries, Diensten (Pro/Contra, Benchmarks)
- Prototypen-Ergebnisse und Lessons Learned
- Machbarkeitsanalysen (Spikes, Proof of Concepts)
- Skalierbarkeits- und Performance-Evaluierung
- Integrationsanalysen (API-Kompatibilität, Datenformate)

## Hinweise

- Technische Entscheidungen mit ADR (`docs/decisions.md`) verknüpfen.
- Benchmark-Ergebnisse und Testdaten hier ablegen, nicht nur Zusammenfassungen.
